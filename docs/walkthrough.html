Setup Walkthrough

1

System Configuration

Enables zswap (compressed RAM cache), expands swap to 8GB, and configures Ollama with performance optimizations for your GTX 1070.

sudo bash ~/sovereign-ai/scripts/system-setup.sh
What this does: Enables zswap with lz4 compression (35% of RAM as compressed cache), expands swap from 4GB to 8GB, sets swappiness to 10, configures Ollama with q8_0 KV cache quantization (halves context memory) and flash attention.
2

Download Models

Pulls the optimal model set. Removes old/weak models first. Total download: ~12 GB.

bash ~/sovereign-ai/scripts/pull-models.sh
SettingValueEffect
KV Cache Typeq8_0Halves VRAM for context window
Flash AttentionOnFaster inference
Keep Alive5 minAuto-unloads idle models
Max Loaded1Prevents VRAM overflow
3

Start the Stack

Launches OpenWebUI (chat interface) and MCPO (MCP tool proxy) via Docker, and verifies Ollama is running.

~/sovereign-ai/start.sh

Then open http://localhost:3000

4

Configure OpenWebUI

First-time setup in the browser:

  • Create your local account at localhost:3000
  • Go to Admin Settings > Connections — verify Ollama URL is http://host.docker.internal:11434
  • Go to Admin Settings > Documents — set embedding model to nomic-embed-text:v1.5
  • Go to Admin Settings > External Tools — add MCP endpoint: http://mcpo:8000
  • Go to Workspace > Knowledge — create collections for your documents
  • Install community tools (see Tools section)
5

VS Code Setup (Continue.dev)

Get AI coding assistance with local models:

RAG Knowledge Base

Building Your Knowledge

This is how your AI learns and refers to your documents:

  • Drop PDFs, markdown, or text files into ~/sovereign-ai/data/rag-docs/
  • In OpenWebUI > Workspace > Knowledge, create a collection
  • Upload files to the collection
  • In chat, click the # icon to attach a knowledge collection
  • The model searches your documents and includes relevant chunks in its context
For textbooks: Create separate collections per subject. Use DeepSeek-R1 for analysis—its chain-of-thought reasoning is excellent for understanding complex material.

Management Commands

~/sovereign-ai/stop.sh

# Update all components
~/sovereign-ai/scripts/update.sh

nvidia-smi
